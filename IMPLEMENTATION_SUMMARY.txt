================================================================================
QUERY PARAMETERS
================================================================================

/api/activities:
  - since (optional): date string (e.g., "2024-01-01")
  - sportType (optional): string (e.g., "Run", "Ride")
  - page (optional): integer (default: 1, min: 1)
  - limit (optional): integer (default: 50, range: 1-100)

/api/stats/monthly:
  - year (optional): 4-digit year (e.g., "2024")
  - sportType (optional): string (e.g., "Run", "Ride")

================================================================================
EXAMPLE USAGE
================================================================================

# Get all activities (paginated)
curl "http://localhost:8000/api/activities"

# Get Run activities since 2024
curl "http://localhost:8000/api/activities?sportType=Run&since=2024-01-01"

# Get 2024 monthly stats
curl "http://localhost:8000/api/stats/monthly?year=2024"

# Get Ride stats for 2024
curl "http://localhost:8000/api/stats/monthly?year=2024&sportType=Ride"
